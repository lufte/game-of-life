## Install
`git clone https://github.com/lufte/game-of-life.git`

`cd game-of-life`

`mix deps.get`

`mix deps.compile` (check https://github.com/lufte/game-of-life.git for requirements for compiling ncurses)

## Run

`iex -S mix`

`Life.run`
